node环境

本地安装node开发环境

React Native 目前需要 NodeJS 8.0 或更高版本

React Native命令行工具（react-native-cli）

React Native 的命令行工具用于执行创建、初始化、更新项目、运行打包服务（packager）等任务。

npm install -g yarn react-native-cli
